Reflection

My learning journey throughout the Mobile Application Programming course.

Throughout this course, I developed my understanding of mobile application development using Android Studio, Kotlin, and Jetpack Compose. The laboratory activities helped me build technical knowledge step by step, starting from basic interface design and progressing towards navigation, database concepts, API integration, and sensor usage.


Through Project 1 and Project 2, I learned how to plan, design, implement, and improve a mobile application based on a real-world sustainability problem. Developing EcoTracker also helped me understand how mobile technology can encourage users to practise more responsible consumption habits in their daily lives.
