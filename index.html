Lab exercises completed throughout the semester.

Projects

Course projects developed throughout the semester.

Project 1: EcoTracker

Initial development of the EcoTracker mobile application.

Project Overview

EcoTracker was developed to support SDG 12 (Responsible Consumption and Production) by encouraging users to adopt environmentally responsible habits in their daily lives.

The application allows users to record sustainable actions, monitor their progress, and learn about the impact of their behaviour on the environment.

Features

  • Action logging system
  • Progress tracking
  • Sustainability awareness
  • Navigation between screens
  • User-friendly interface
View Project 1 GitHub

Project 2: EcoTracker Advanced Version

Enhanced application with cloud, API and sensor integration.

Project Overview

Project 2 expanded the functionality of EcoTracker by integrating advanced Android development concepts such as Firebase Firestore, REST API services, local persistence, and mobile hardware sensors.

The goal was to create a more interactive and practical application that promotes sustainable consumption habits while meeting the requirements of SDG 12.

Advanced Features

  • Firebase Firestore integration
  • REST API integration for live environmental information
  • Camera sensor implementation
  • Community sharing functionality
  • Image gallery support
  • Environmental impact tracking
  • Multi-screen navigation flow

Technologies Used

  • Kotlin
  • Jetpack Compose
  • Firebase Firestore
  • Retrofit
  • REST API
  • Android Camera Sensor
  • GitHub

Reflection

My experience throughout the Mobile Application Programming course.

Throughout this course, I progressively developed my mobile application development skills, starting from basic user interface design and advancing towards more complex concepts such as navigation, local data persistence, API integration, cloud services, and sensor implementation.


The laboratory activities provided valuable hands-on experience that helped strengthen my understanding of Android development using Kotlin and Jetpack Compose. Each activity introduced new concepts that were later applied in larger projects.


The development of EcoTracker enhanced my problem-solving abilities, debugging skills, and confidence in building mobile applications that address real-world sustainability issues aligned with SDG 12.
